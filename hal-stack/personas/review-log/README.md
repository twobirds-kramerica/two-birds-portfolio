---

## Rules

- Every Stage 3+ sprint MUST have a review log entry
- Stage 1-2 sprints: review log optional (Drew decides)
- REWORK verdicts create a BLOCKED status in sprint-queue.md
- Aaron overrides are valid but permanently logged
- Review files are never deleted or modified after the sprint ships
