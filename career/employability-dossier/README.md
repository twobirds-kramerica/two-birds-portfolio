# Employability Dossier

Raw material for building a CV and pursuing employment. Does NOT contain a CV. Contains the evidence, research, and questions needed to build the right CV for the right market.

## Files

| File | Purpose |
|------|---------|
| `skills-inventory.md` | 9 skills with evidence from repo history. What Aaron can and cannot claim. |
| `gap-analysis.md` | Honest gaps, what closes each gap, and common misconceptions. |
| `short-term-gig-research.md` | 7 platforms evaluated. Top 3: Outlier, Surge AI, Scale AI. |
| `full-time-target-roles.md` | 5 role types researched with real postings. Top fit: Product Operations. |
| `cv-interview-questions.md` | 29 questions for Aaron to answer before the CV gets written. |

## How Aaron Should Use This

1. Read `gap-analysis.md` first (5 min). Understand what's strong and what's missing.
2. Answer `cv-interview-questions.md` (15-20 min). Multiple choice where possible.
3. Decide urgency: gig platform (days) vs full-time search (months) vs both.
4. If gig: sign up for Outlier AI. Takes 1-2 weeks to first paid task.
5. If full-time: target Product Operations roles at AI companies. Check Cohere (Toronto).
6. Bring the answered questions to the next Claude Code session. The CV gets built from the answers.

## What This Is NOT

- Not a CV. The CV is built AFTER Aaron answers the questions.
- Not a job application. No applications submitted.
- Not a commitment to a direction. Aaron decides.
