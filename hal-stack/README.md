# HAL Stack — Two Birds Innovation
Always-on AI development and automation server.

## What HAL Is
HAL is the persistent infrastructure layer that makes Two Birds Innovation autonomous.
It runs on the i5 Lenovo (and eventually the home desktop) as an always-on server.
Aaron can dial in from any device — phone, new laptop, tablet — and have a full
Claude Code session running on a powerful stationary machine.

## Core Principles
- MODULAR: every component works independently
- REDUNDANT: critical paths have fallbacks
- FREE: open source wherever possible
- SCALABLE: solo use now, team or enterprise later
- API-FIRST: every service exposes an API
- PAYWALL-AWARE: free tiers or self-hosted alternatives used throughout
- BUSINESS-SEPARABLE: each major process could become its own product
